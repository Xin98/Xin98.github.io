找实习的这段日子

发表于 2020-05-20 更新于 2020-05-25 分类于 生活 阅读次数： Valine：
本文字数： 3.6k 阅读时长 ≈ 3 分钟

  大家好，一个多月没见了，有没有想我？在这个特殊的日子里，更新一篇博客聊以慰藉hhhh。最近忙于找实习，博客这边不知不觉就荒废了一个多月了。好在最后的结果还不错，博客以后会慢慢恢复更新的！

实习offer情况

  从2月底开始准备找实习，历经了2个多月。前期(2月底-3月底)主要是学习，中期(3月底-4月底)开始面试，期间因为知识准备的不充分，以及缺少面试经验（面试的时候超级紧张）碰壁无数，一度觉得自己可能找不到实习了。但是好在虽然面试结果不好，但自己也在一直去努力的学习，准备。最后在后期（4月底-5月中旬）拿到了offer。

offer情况：

  • 阿里新零售供应链java开发(已发意向书)
  • 京东基础交易平台java后台开发(offer待审批) – 5-20更新(审批通过了，刚刚收到录取邮件)

这段时间我面了哪些？

  最开始面试的就是腾讯，原因就是我朋友和我说，腾讯早面，早凉，早被捞(2333)，在家的学习效率不算高，所以一直觉得自己准备的不够充分，也非常焦虑，听了朋友的建议，就投了一下腾讯，开始了自己的找实习的第一次面试。虽然凉了，但是面试了一次后，对于面试的那种焦虑也得到了缓解。
  在这边我也建议大家，能早投就早投，很多公司春招秋招都会有提前批，一般提前批是不记录到系统的，如果你之前面试经验不足，提前批的面试真的是个锻炼自己的非常好的机会，要把握住。这段时间面了来来回回差不多20次，面试到了后面，感觉就像是和面试官在聊天一样了，聊得好就大概率会过，也不会像最开始面试那样紧张的说话都会结巴。
  除了腾讯，之后面了华为、百度、美团、京东、阿里。战绩如下：

  • 腾讯(5捞5挂，腾讯后面问面试官说大概是3轮技术+1轮hr) ×
  • 华为(1面就过了，部门比较偏，后面就放弃了) √
  • 美团(1面40多min，2面1h10min，收到感谢信) ×
  • 百度(效率很高，一下午就把面试安排搞完了1面50min, 2面50min挂，百度应该是有3面的) ×
  • 京东(1面30min，2面30min，hr面 17min) 待√
  • 阿里(1面45min，2面1h5min，3面30min，交叉面13min，hr面29min) √

  给大家说一个我比较心酸时刻，当时面完阿里hr后，第二天问内推人面试情况，被内推人告知没有hc了。听到这个结果的我当时心态大崩，好在后面和部门leader沟通，leader叫我再等等，应该还有hc，还好运气好，等到了。这件事给我的感受就是，大家找工作啊实习啊，能早投就早投，投晚了，面试过了，没hc真的会巨难受。
  还有做了一些公司的笔试，可能是没做好，也有可能是投递的时间太晚了，没有收到面试通知。还有投字节的话最好是找人内推，我之前投的一个部门，没有内推，过了1个月后从筛选状态变成了部门面试已经结束，简直太坑了！

面经

  这一部分，可能会比较少，当时刚开始面试的时候，我还比较积极，面完了立刻就写文档把我遇到的题目都会记录下来，后面面试的次数多了，懒了起来，就没记录了，所以很多公司的面试问题都忘了，不过关于java岗会问的一些方面我还是总结了一下，大家可以参考。首先上我记录的面经吧。

腾讯ieg后台开发（初试）

  1. 自我介绍.
  2. 熟悉java基础吗？熟悉哪些呢？容器比较熟悉
  3. ArrayList 和 linkedlist 的区别，以及使用场景。
  4. HashMap的底层实现？为什么要用红黑树？什么时候用链表什么时候用红黑树？
  5. 并发hashmap的源码有看过吗？
  6. Synchronize有了解过吗？我第一次没听清，就答的不了解。
  7. 其他锁有了解吗？wait notify的实现有了解吗？什么情况下才能用wait呢？
  8. aqs的源码有看过吗？怎么实现的知道吗？
  9. Spring ioc/aop？
  10. 两台机器怎么实现持久化登录呢？
  11. 数据库事务有了解吗？ACID单机一定能满足吗？哪些不必须呢？事务的传播级别？
  12. 数据库的隔离级别？以及各个级别会对应什么问题？
  13. 数据库索引有了解吗？使用场景？哪些sql不能有效的利用索引？怎么判断有没有使用索引？

  4. 小说怎么去分类？

腾讯ieg后台开发(最近的一次被捞 50min)

  一开始就和我说他们是做游戏相关的，需要c++。而我是java，不过还是聊了有50min。最后问他如果要进他这里需要准备什么？(准备c++)，所以给我的感觉就是，大家如果想进腾讯，c++大法好啊！

  1. 自我介绍
  2. java容器HashMap数据结构
  3. 并发模块ConcurrentHashMap的实现（说了一下jdk1.7和jdk1.8的实现）
  4. 知道哪些并发模块
    (CountDownlatch, Semephore, CyclicBarrier)
  5. MySql数据库引擎有哪些？
    （innodb，myisam）
  6. Mysql索引用的什么数据结构？为什么？
    （b+树，回答了b+树和b树的区别）
  7. 进程和线程的区别是什么？
  8. 多线程的优势是什么？
    （从单核和多核展开回答的）
  9. Linux有用过吗？常见的一些命令有哪些？知道哪些查看性能命令？
    （答了top， nvidia-smi）
    top有哪些性能指标？
    （回答了cpu占有率，内存占用情况）
  10. tcp三次握手？为什么要3次？
  11. tcp4次挥手？为什么要4次？timewait的作用？
  12. epoll知道吗？（不知道）
  13. udp发3个包，接收方有可能接收到几个包？
  14. 为什么网络时延大了，tcp可靠性会降低？
  15. C++模板的作用？
  16. c++与java的区别？
  17. java的模板（泛型）是怎么实现的？
  18. 有什么想问我的？

百度一面 (50min)

  1. 自我介绍
  2. LinkedList有20个数据，底层是如何得到索引为11的节点的？
    （LinkedList是双向链表，11 > 20/2，从后往前遍历得到索引为11的节点）
  3. HashMap的数据结构
  4. TreeMap的实现数据结构，以及他的继承关系
    (TreeMap – NavigableMap – SortedMap – Map)
  5. 数据库索引什么时候会失效，如何解决？
  6. 后面问了一下项目记不太清了

阿里三面（部门leader面 30min）

  1. 简单介绍自己
  2. 主要问了设计模式，各个设计模式在java中的哪些地方用了。
  3. 给一个无限长的数组（Object数组），两种不同类型的Object,
    key Object 和 value Object 如何实现一个hash表？
    (我回答计算了key在Object的idx后，然后把key放入奇数位置，
    value放入相应的偶数位置，就可以实现key查找value的功能)
